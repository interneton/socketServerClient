# 핑 패킷 전달

`NetworkManager.cs`에서 핑 패킷 전달 기능을 추가하였습니다. 이 코드는 클라이언트와 서버 간의 지연 시간을 측정하고 연결을 유지하기 위해 핑 메시지를 처리합니다.

### 핑 패킷 처리 단계

1. **핑 패킷 수신**:
   - `ProcessReceivedData` 함수는 수신한 패킷의 유형을 확인합니다. `Ping` 패킷이 수신되면 `HandlePingPacket` 함수가 호출됩니다.

2. **퐁 패킷 전송**:
   - `HandlePingPacket` 함수는 패킷을 역직렬화하여 타임스탬프를 추출합니다.
   - 이후 `SendPong` 함수를 호출하여 타임스탬프를 서버로 다시 전달함으로써 지연 시간을 계산할 수 있도록 합니다.

### 관련 함수

- **HandlePingPacket**:
  - 수신된 `Ping` 패킷을 추출 및 처리합니다.
  - `SendPong`을 호출하여 서버로 원래 타임스탬프를 포함한 응답을 전송합니다.

- **SendPong**:
  - 타임스탬프를 포함한 `Pong` 메시지를 생성합니다.
  - 헤더와 직렬화된 메시지를 포함한 패킷을 생성하여 서버로 전송합니다.

이 핑-퐁 통신은 연결 상태와 지연 시간을 지속적으로 모니터링하는 데 필요한 하트비트 패킷을 제공하여, 클라이언트와 서버 간의 연결을 유지합니다.

## 주요 코드 섹션

- `HandlePingPacket(byte[] data)`
- `SendPong(long lastTimestamp)`
